## 🎯 核心理念

"蚁元"寓意：
- **蚁**：像蚂蚁一样简单、协作的 Agent 单元，它们按照不同职责分类（如同蚁后、工蚂、雄蚁、飞蚁等）
- **元**：元级（Meta）协调，基础单元构建复杂系统

单个蚂蚁简单，蚁群却展现出惊人的集体智慧。
每个代理就像一只蚂蚁，简单但协作产生复杂能力。

## 🏗️ 架构特点

- ✅ **进程隔离**：基于 OpenClaw sessions_spawn，完全安全隔离
- ✅ **角色系统**：继承 Agency 153+ 专业角色
- ✅ **零代码风险**：无自动代码执行，纯指令协调
- ✅ **记忆集成**：与 OpenViking 四层记忆架构无缝集成
